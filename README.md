## Pokyny

* ### Download dependencies

    V root Adresári príkaz:

    ```bash
    npm install
    ```

* ### Spustenie servera

    ```bash
    npm start
    ```

* ### Build webappky

    [Comming soon]

    ```bash
    npm build
    ```

### Prerequisites

![NodeJS](https://img.shields.io/badge/NodeJS-v8.11.4-blue.svg)
