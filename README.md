## Pokyny

* ### Download dependencies

    V root Adresári príkaz:

    ```npm
    npm
    ```

* ### Spustenie servera

    ```npm
    npm start
    ```

* ### Build webappky

    [Comming soon]

    ```npm
    npm build
    ```
